# Valam Trust — NGO Web Application

## Overview
**Valam Trust** is a highly polished, responsive Single-Page Application (SPA) built with purely custom HTML, CSS, and JavaScript that serves as the modern digital front door for a non-profit organization. It features a premium, bespoke UI utilizing dynamic CSS grid layouts, smooth JavaScript-powered routing, and high-quality visuals to elegantly showcase the foundation’s impact, history, and community focus. The platform drives engagement through seamlessly integrated modules, including interactive multi-step donation workflows, volunteer application funnels, and tiered corporate CSR partnership layouts.

## Features
- **Dynamic Single-Page Routing:** Instant, seamless navigation between application views (`Home`, `About`, `Our Work`, `Get Involved`, `Donate`, `News`) without full page reloads.
- **Premium Custom UI:** A unique, polished design aesthetic using glassmorphism overlays, robust CSS variables for thematic consistency, and perfectly responsive grid/flexbox architecture.
- **Interactive Timeline Layout:** A fully responsive and precisely aligned timeline system tracking the organization's legacy.
- **Comprehensive Donate Engine:** Clean multi-tier donation UI supporting varying payment frequencies, visual progress data on fund allocation, and transparent impact metrics. 
- **Engagement Funnels:** Detailed sections providing easy-access forms for volunteer applications and distinct tiered packages for corporate CSR partnerships.

## Tech Stack
- **Structure:** Semantic HTML5
- **Styling:** Vanilla CSS3 (Custom Properties, CSS Grid, Flexbox, Keyframe Animations)
- **Logic:** Vanilla JavaScript (ES6+ for virtual DOM transitions, dynamic routing across components, and interactive UI toggles)
- **Typography:** `Yeseva One`, `Lora`, `Nunito Sans` (via Google Fonts)

## Core File Structure
```text

├── valam-trust.html  # Main HTML scaffolding, content, and layout structure
```

## Getting Started
To view, interact with, and run this application locally, there are zero complex frontend dependencies required:
1. Clone or download this repository to your local development machine.
2. Open the `valam-trust.html` file directly in any modern web browser.
3. (Optional) Run via a local dev server like VS Code's `Live Server` for an optimal development experience.
